# 2. BACKEND ENDPOINTS
mkdir api/routes/developer
touch api/routes/developer/{new_key.py,usage.py,sandbox.py}

# new_key.py
- Generate UUID4 key, store hashed in DB.
- Assign default tier “Sandbox”.
- Limit one active key per account.

# usage.py
- Return call count, quota, renewal date.

# sandbox.py
- Mirror production API with mock data.
- Enforce max 100 calls/day.

Security:
- Header “x-api-key”
- Rate-limit middleware.
