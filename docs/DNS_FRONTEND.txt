DNS Configuration for Levqor Frontend Custom Domain
=====================================================

FRONTEND DOMAIN: app.levqor.ai
DEPLOYMENT: Vercel

DNS RECORDS REQUIRED (Cloudflare):
-----------------------------------

Record Type: CNAME
Name: app
Value: cname.vercel-dns.com
Proxy Status: OFF (DNS Only - Orange cloud disabled)
TTL: Auto

⚠️ IMPORTANT: Proxy must be OFF for Vercel to provision SSL certificate

Alternative (if not using Cloudflare):
--------------------------------------
Record Type: CNAME
Name: app
Value: cname.vercel-dns.com
TTL: 3600

VERCEL CONFIGURATION:
---------------------

1. Add Custom Domain in Vercel Dashboard:
   Project → Settings → Domains → Add Domain
   Enter: app.levqor.ai
   
2. Vercel will provide domain verification instructions

3. Wait for SSL certificate provisioning (usually 1-5 minutes)

ENVIRONMENT VARIABLES (Vercel Project Settings):
-------------------------------------------------
Set these in: Project → Settings → Environment Variables

Production Environment:
  NEXT_PUBLIC_BACKEND_BASE=https://api.levqor.ai
  NEXT_PUBLIC_BACKEND_CHECKOUT=https://api.levqor.ai/billing/create-checkout-session
  NEXT_PUBLIC_BACKEND_SUMMARY=https://api.levqor.ai/api/v1/marketing/summary
  NEXT_PUBLIC_ASSETS_BASE=https://api.levqor.ai
  NEXT_PUBLIC_APP_NAME=Levqor
  DASHBOARD_TOKEN=<your-secret-token>

VERIFICATION:
-------------
After DNS propagation and Vercel setup:

1. Test DNS resolution:
   $ dig app.levqor.ai CNAME
   $ nslookup app.levqor.ai

2. Test HTTPS connectivity:
   $ curl -I https://app.levqor.ai/
   Expected: HTTP/2 200

3. Test API connectivity:
   $ curl -I https://api.levqor.ai/health
   Expected: HTTP/2 200

4. Test CORS:
   $ curl -H "Origin: https://app.levqor.ai" -I https://api.levqor.ai/api/v1/marketing/summary
   Expected: Access-Control-Allow-Origin: https://app.levqor.ai

KEY PAGES TO VERIFY:
--------------------
  - https://app.levqor.ai/ (landing page)
  - https://app.levqor.ai/pricing (pricing page)
  - https://app.levqor.ai/dashboard?token=<token> (analytics)

DEPLOYMENT STEPS:
-----------------

1. Deploy to Vercel:
   $ cd levqor-web
   $ vercel --prod
   
   Or connect GitHub repo to Vercel for auto-deployment

2. Configure custom domain in Vercel dashboard

3. Add DNS records in Cloudflare (CNAME with Proxy OFF)

4. Wait for SSL certificate (automatic via Vercel)

5. Set environment variables in Vercel project settings

6. Redeploy after adding environment variables

TROUBLESHOOTING:
----------------

Issue: "Invalid SSL Certificate"
Solution: Ensure Cloudflare proxy is OFF (DNS Only mode)

Issue: "Domain not found"
Solution: Wait for DNS propagation (up to 48h, usually 5-30 min)

Issue: "CORS errors"
Solution: Verify backend CORS includes https://app.levqor.ai

Issue: "Stripe checkout fails"
Solution: Check NEXT_PUBLIC_BACKEND_CHECKOUT environment variable

For Vercel-specific deployment help, visit:
