EchoPilotAI API Documentation

RESTful API reference for EchoPilotAI automation platform.

Health & Status

GET /health

Check system health status

{
  "status": "healthy",
  "timestamp": 1234567890
}

Feedback

POST /api/feedback

Submit user feedback

Request:
{
  "email": "user@example.com",
  "message": "Great product!",
  "page": "https://echopilotai.com/pricing"
}

Response:
{
  "ok": true,
  "message": "Feedback received"
}

Incident Management

POST /api/incident/report

Trigger incident report (requires authentication)

Headers:
Authorization: Bearer <ADMIN_BEARER>

Response:
{
  "ok": true,
  "message": "Incident report generated"
}

Multi-Tenant

GET /api/tenant/ping

Get current tenant information

Headers:
X-Tenant: acme

Response:
{
  "tenant": "acme",
  "ok": true
}

Billing Analytics

GET /api/billing/summary

Get billing analytics summary

Response:
{
  "mrr": 12500.00,
  "arpa": 125.00,
  "churn": 2.5,
  "customers": 100
}

Usage Metering

POST /api/meter/usage

Record usage for metered billing

Request:
{
  "meter": "api_calls",
  "quantity": 100,
  "user": "user123",
  "tenant": "acme"
}

Response:
{
  "ok": true
}

AI Cost Management

GET /api/ai/cost-cap

View AI cost budget and remaining allocation

Response:
{
  "daily_cap_usd": 2.00,
  "spent_today_usd": 0.45,
  "remaining_usd": 1.55,
  "ok": true
}

Autopilot

POST /api/autopilot/toggle

Enable or disable autopilot mode

Request:
{
  "enabled": true
}

Response:
{
  "enabled": true,
  "ok": true
}
GET /api/autopilot/status

Get autopilot status

Response:
{
  "enabled": false
}
